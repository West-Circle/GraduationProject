**题目**：基于Spring Boot与Vue的运动物品商城设计与实现

技术：前端Vue框架，后端Spring Boot + Mybatis Plus框架，数据库MySQL

项目特点：实现商城网站的基本功能，主要的功能特点是基于众包鉴伪的实现。
